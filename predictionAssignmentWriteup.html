Executive Summary

This report describes an analysis of a weight lifting data set described in the following journal article:

Velloso, E.; Bulling, A.; Gellersen, H.; Ugulino, W.; Fuks, H. Qualitative Activity Recognition of Weight Lifting Exercises. Proceedings of 4th International Conference in Cooperation with SIGCHI (Augmented Human 13). Stuttgart, Germany: ACM SIGCHI, 2013.

that contains measurements from accelerometers placed on the:

  • weight lifted (i.e.dumbell)
  • belt
  • forearm
  • arm

for six test participants. The results presented in this document will illustrate the process that I followed to:

1.) Load & clean the training and test data
2.) Perform exploratory data analysis
3.) Train a prediction algorithm (i.e.random forest)
4.) Estimate the out-of-sample error
5.) Apply the trained predictor to the test data set

The accuracy of the predictor that I will describe in this report is 70% based on scoring results generated by the Coursera website. My analysis of this ressult is that it is slightly lower than the average accuracy of 78.8% estimated from the normalized confusion matrix shown on the bottom of the website that includes a description of this weight lifting dataset.

loadExerciseData <- function(exerciseDataFile) {
#------------------------------------------------------------------------
# Loads a *.csv that contains training or testing data into memory.
# The operations performed by this function include:
# - Removing "_"'s from variable names
# - Removing "."'s from variable names
# - Combing the rawtimestamppart1 & rawtimestamppart2 variables into
#   a timestamp variable that represents the measurement time in the
#   Unix (i.e. "seconds since the epoch") format
#
# Args:
#   exerciseDataFile: String that stores the data file name
#
# Returns:
#   exerciseData: Data frame that contains weight lifting data
#
# Constraints:
#   This function assumes that the weight lifting data file is located
#   in the following relative path: ./Data
#------------------------------------------------------------------------
    if (!file.exists("./Data")) {
        dir.create("./Data")
        
        trainingDataURL <- 
            paste0("https://d396qusza40orc.cloudfront.net/predmachlearn",
                   "/pml-training.csv")
    
        testDataURL <-
            paste0("https://d396qusza40orc.cloudfront.net/predmachlearn",
                   "/pml-testing.csv")
        
        download.file(trainingDataURL, destfile = "./Data/trainingData.csv")    
        download.file(testDataURL, destfile = "./Data/testData.csv")
    }

    exerciseData <- read.csv(file = file.path("./Data",
                                              exerciseDataFile),
                             header=TRUE,
                             stringsAsFactors = FALSE)

    colnames(exerciseData) <- tolower(colnames(exerciseData))
    variableNames <- colnames(exerciseData)

    for (i in seq_len(length(variableNames))) {
        variableNames[i] <- gsub("_", "", variableNames[i])
        variableNames[i] <- gsub("\\.", "", variableNames[i])
    }
    
    colnames(exerciseData) <- variableNames
    
    exerciseData$timestamp <- exerciseData$rawtimestamppart1 + 
                              exerciseData$rawtimestamppart2*1E-6
    
    exerciseData <- 
        exerciseData[,!colnames(exerciseData) %in% c("x",
                                                     "rawtimestamppart1",
                                                     "rawtimestamppart2")]

    return(exerciseData)
}

The number of rows in the training and test data is 19622 and 20, respectively. The third step that I performed to prepare this data for analysis is transforming it into a tidy data. The key characteristic of this data transform is that the data set variables are contained a data frame columns and there is one observation per data frame row.

The purpose of initializeDataWindowIndex(), which is the first R function defined in the following code chunk, is to aggregate the time series data associated a specific individual and weight lifting motion class. Next, the second function defined in the following section of R software, computeDataWindowCount(), computes the number of time series window for each dumbbell exercise motion. Simiarily, computeDataWindowSize() computes the size of each time series associated with a weight lifting motion class. Finally, computeDataWindowSizeSummary() computes a six-number summary:

  • Minimum value
  • 1st quantile
  • Median
  • Mean
  • 3rd quantile
  • Maximum

of the data window size for each dumbbell exercise motion. The rationale for writing these functions is that they simplified an initial exploratory analysis of the data.

## DEFINE FUNCTIONS

initializeDataWindowIndex <- function(trainingData,
                                      minLength = 0,
                                      maxLength = NA) {
#------------------------------------------------------------------------
# Initializes a compound list that contains the data frame row indices
# corresponding to all time series windows in a weight lifting data set 
#
# Required Args:
#   trainingData: Data frame that stores the training data
#
# Optional Args:
#   minLength: Integer that specifies the minimum number of data window 
#              samples
#
#   maxLength: Integer that specifies the maximum number of data window 
#              samples
#
# Returns:
#   dataWindow: Compound list that contains the data frame row indices
#               corresponding to all time series windows in a weight 
#               lifting data set
#------------------------------------------------------------------------
    weightLiftingClasse <- unique(trainingData$classe)
    
    dataWindow <- list();
    for (curClasse in weightLiftingClasse) {
        dataWindow[[curClasse]] <- list();    
    }
    
    startOfNewWindow <- which(trainingData$newwindow == "yes")

    numDataWindows <- length(startOfNewWindow)
    
    for (n in seq(0, numDataWindows)) {
        if (n == 0) {
            curWindowIdx <- 1:(startOfNewWindow[1] - 1)
        }
        else if (n == numDataWindows) {
            curWindowIdx <- startOfNewWindow[n]:nrow(trainingData)
        }
        else
        {
            curWindowIdx <- startOfNewWindow[n]:(startOfNewWindow[n + 1]- 1)
        }
        curClasse <- trainingData$classe[curWindowIdx[1]]
    
        minLengthTest <- length(curWindowIdx) >= minLength

        if (is.na(maxLength)) {
            maxLengthTest <- TRUE
        }
        else {
            maxLengthTest <- length(curWindowIdx) <= maxLength
        }

        if (minLengthTest & maxLengthTest) {
            dataWindow[[curClasse]][[length(dataWindow[[curClasse]]) + 1]] <- 
                curWindowIdx            
        }
    }
    
    return(dataWindow)
}

computeDataWindowCount <- function(dataWindow) {
#------------------------------------------------------------------------
# Computes the number of time series windows for each measurement classe
#
# Args:
#   dataWindow: Compound list that contains the data frame row indices
#               corresponding to all time series windows in a weight 
#               lifting data set
#
# Returns:
#   dataWindowCount: Data frame that stores number of time series windows 
#   for each measurement classe
#------------------------------------------------------------------------
    dataWindowCount <- data.frame()
    for (curClasse in names(dataWindow)) {
        dataWindowCount <- rbind(dataWindowCount,
                                 data.frame(classe=curClasse,
                                            count=length(dataWindow[[curClasse]])))
    }

    dataWindowCount$classe <- as.character(dataWindowCount$classe)
    
    return(dataWindowCount)
}

#------------------------------------------------------------------------
# Computes the size of each time series window contained in a weight
# lifting data set
#
# Args:
#   dataWindow: Compound list that contains the data frame row indices
#               corresponding to all time series windows in a weight 
#               lifting data set
#
# Returns:
#   dataWindowSize: Data frame that stores the size of each time series
#   contained in a weight lifting data set
#------------------------------------------------------------------------
computeDataWindowSize <- function(dataWindow) {
    dataWindowSize <- data.frame()

    for (curClasse in names(dataWindow)) {
        for (curIdx in dataWindow[[curClasse]]) {
            dataWindowSize <- 
                rbind(dataWindowSize,data.frame(classe=curClasse,
                                                windowsize=length(curIdx)))
        }
    }
    
    return(dataWindowSize)
}

computeDataWindowSizeSummary <- function(dataWindowSize) {
#----------------------------------------------------------------------
# Generates a data frame that stores a six number summary (i.e. 
# minimum, 1st quantile, median, mean, 3rd quantile, and maximum)
# of time series window for each weight lifting measurement classe
#
# Args:
#   dataWindowSize: Data frame that stores the size of each time series
#   contained in a weight lifting data set
#
# Returns:
#   dataWindowSizeSummary: data frame that stores a six number summary 
#   (i.e. minimum, 1st quantile, median, mean, 3rd quantile, and 
#   maximum) of time series window for each weight lifting measurement 
#   classe
#----------------------------------------------------------------------
    dataWindowSizeSummary <- data.frame()
    for (curClasse in unique(dataWindowSize$classe)) {
        curCounts <- subset(dataWindowSize,
                            dataWindowSize$classe==curClasse)
    
        curSummary <- summary(curCounts$windowsize)
    
        dataWindowSizeSummary <- 
            rbind(dataWindowSizeSummary,
                  data.frame(classe=curClasse,
                             min=curSummary[1],
                             firstquantile=curSummary[2],
                             median=curSummary[3],
                             mean=curSummary[4],
                             thirdquantile=curSummary[5],
                             max=curSummary[6]))
    }
    rownames(dataWindowSizeSummary) <- NULL
    
    dataWindowSizeSummary$classe <- as.character(dataWindowSizeSummary$classe)    
    
    return(dataWindowSizeSummary)
}

processNumericVariables <- function(curDataWindowIn,
                                    numericVariables) {
#----------------------------------------------------------------------
# Computes the mean of each time series numeric variable associated
# with a specific individual and dumbbell exercise motion class.
#
# ArgS:
#   curDataWindowIn: Data frame that contains the scalar and numeric 
#                    time series measurements associated with a 
#                    specific individual and dumbbell exercise motion 
#                    class.
#
#   numericVariables: Character vector that stores the names of time 
#                     series variables
#
# Returns:
#   curDataWindow: 1 row data frame that contains the mean of each time
#                  series numeric variable associated with a specific 
#                  individual and dumbbell exercise motion class.
#----------------------------------------------------------------------
    numNumericVariables <- length(numericVariables)
    
    for (i in seq_len(numNumericVariables)) {
        curSamples <- curDataWindowIn[,numericVariables[i]]
        
        numValidSamples <- sum(!is.na(curSamples))
        
        if (numValidSamples == 1) {
            curNumericVariable <- 
                data.frame(curSamples[!is.na(curSamples)])
        }
        else {
            curNumericVariable <- data.frame(mean(curSamples))
        }
        colnames(curNumericVariable) <- numericVariables[i]
     
         if (i == 1) {
             curDataWindow <- curNumericVariable
         }
         else {
             curDataWindow <- cbind(curDataWindow, curNumericVariable)
         }
    }
    
    return(curDataWindow)
}

processCharacterVariables <- function(curDataWindowIn,
                                      characterVariables) {
#----------------------------------------------------------------------
# Converts a scalar measurement associated with a specific individual 
# and dumbbell exercise motion class to a numeric value.
#
# ArgS:
#   curDataWindowIn: Data frame that contains the scalar and numeric 
#                    time series measurements associated with a 
#                    specific individual and dumbbell exercise motion 
#                    class.
#
#   characterVariables: Character vector that stores the names of scalar 
#                       variables
#
# Returns:
#   curDataWindow: 1 row data frame that contains the numeric value of
#                  each sclar variable associated with a specific 
#                  individual and dumbbell exercise motion class.
#----------------------------------------------------------------------
    numCharacterVariables <- length(characterVariables)
    
    #---------------------------------------------------------
    for (i in seq_len(numCharacterVariables)) {
        curVar <- characterVariables[i]
        curSamples <- suppressWarnings(as.numeric(curDataWindowIn[,curVar]))
        
        validIdx = which(!is.na(curSamples))
        
        numValidSamples <- length(validIdx)
        
        if (numValidSamples > 0) {
            curCharacterVariable <- data.frame(median(curSamples[validIdx]))
        }
        else {
            curCharacterVariable <- data.frame(NA)
        }
        colnames(curCharacterVariable) <- characterVariables[i]

        if (i == 1) {
            curDataWindow = curCharacterVariable
        }
        else {
            curDataWindow <- cbind(curDataWindow, curCharacterVariable)    
        }
    }
    
    return(curDataWindow)
}

initializeMeasurements <- function(trainingData,
                                   dataWindow) {
#----------------------------------------------------------------------
# Initializes a compound list that stores the following attributes:
#
# - cleanTrainingData: Data frame where each row contains the
#                      measurements associated with a specific 
#                      individual and dumbbell exercise motion
#
# - cleanDataWindow: Compound list that contains the data frame row 
#                    indices corresponding the clean training data 
#                    measurements
#
# - ambiguousClasseDataWindow: Compound list that contains the data 
#                              frame row indices corresponding the 
#                              time series windows where the dumbbell
#                              exercise motion class was ambiguous
#
# - characterVariables: Character vector that stores the name of
#                       scalar variables associated with a specific
#                       individual and dumbbell exercise motion
#
# - numericVariables: Character vector that stores the name of
#                     numeric time series variables associated with 
#                     a specific individual and dumbbell exercise 
#                     motion
#           
# Args:
#   trainingData: Data frame that stores the raw training data
#
#   dataWindow: Compound list that contains the data frame row indices
#               corresponding to all time series windows in a weight 
#               lifting data set
#
# Returns:
#   Compound list that stores clean training data and data cleaning 
#   debug parameters
#----------------------------------------------------------------------
    cleanData <- list()
    ambiguousClasseDataWindow <- list()
    incompleteCaseDataWindow <- list()
    cleanDataWindow <- list()
    cleanTrainingData <- data.frame()

    characterVariables <- vector()
    numericVariables <- vector()
    
    for (i in seq_len(length(variableNames))) {
        curClass <- class(trainingData[,variableNames[i]])
        if (curClass == "character") {
            characterVariables <- c(characterVariables, variableNames[i])
        }
        else if (curClass == "numeric" | curClass == "integer") {
            numericVariables <- c(numericVariables, variableNames[i])
        }
    }
    
    for (curClasse in names(dataWindow)) {
        dataWindowNumber <- 1
        
        for (curDataWindowIdx in dataWindow[[curClasse]]) {
            curDataWindowIn <- trainingData[curDataWindowIdx,]
            
            numUniqueClasse <- length(unique(curDataWindowIn$classe))
        
            #-----------------------------------------------------------------
            if (numUniqueClasse == 1) {
                curDataWindow <- processNumericVariables(curDataWindowIn,
                                                         numericVariables)
    
                curDataWindow <- 
                    cbind(curDataWindow,
                          processCharacterVariables(curDataWindowIn,
                                                    characterVariables))

                curDataWindow$datawindownumber <- dataWindowNumber
                
                curDataWindow$windowsize <- length(curDataWindowIdx)
            
                curDataWindow$classe <- curDataWindowIn$classe[1]

                n <- length(cleanDataWindow[[curClasse]]) + 1
                
                cleanDataWindow[[curClasse]][[n]] <- curDataWindowIdx 
                
                cleanTrainingData <- rbind(cleanTrainingData,
                                           curDataWindow)
    
                dataWindowNumber <- dataWindowNumber + 1
            }
            #-----------------------------------------------------------------
            else {
                n <- length(ambiguousClasseDataWindow[[curClasse]]) + 1

                ambiguousClasseDataWindow[[curClasse]][[n]] <- 
                    curDataWindowIdx 
            }
        }
    }
    
    cleanData[[1]] <- cleanTrainingData
    cleanData[[2]] <- cleanDataWindow
    cleanData[[3]] <- ambiguousClasseDataWindow
    cleanData[[4]] <- characterVariables
    cleanData[[5]] <- numericVariables
    
    return(cleanData)
}

computePercentNAValues <- function(predictionData) {
#----------------------------------------------------------------------
# Determines the percent of NA values for each variable in a training
# or test dat set.
#
# Args:
#   predictionData: Data frame that contains training or test data
#
# Returns:
#   percentNAValues: Data frame that describes percent of NA values 
#   for each variable in a training or test dat set.
#----------------------------------------------------------------------
    variableNames <- colnames(predictionData)
    
    percentNAValues <- vector('numeric',length(variableNames))

    for (i in seq_len(length(percentNAValues))) {
        percentNAValues[i] <- 
            sum(is.na(predictionData[,variableNames[i]]))
    }

    percentNAValues <- 100 * (percentNAValues / nrow(predictionData))
    percentNAValues <- data.frame(percentnavalues=percentNAValues)
    percentNAValues$variable <- variableNames
    
    return(percentNAValues)
}

computeNAValueStatistics <- function(percentNAValues) {
#----------------------------------------------------------------------
# Computes the frequency of each % NA values region, which is a factor
# variable generated by the cut2() function applied to the continuous
# percentnavalues variable
#
# Args:
#   percentNAValues: Data frame that describes percent of NA values 
#   for each variable in a training or test dat set.
#
# Returns:
#   percentNAValuesRegions: Data frame that describes the frequency of 
#   each % NA values region, which is a factor variable generated by 
#   the cut2() function applied to the continuous percentnavalues 
#   variable.
#----------------------------------------------------------------------
    percentNAValuesRegions <- as.data.frame(table(percentNAValues$region))
    
    colnames(percentNAValuesRegions) <- c("region",
                                          "frequency")
    
    percentNAValuesRegions$region <- 
        as.character(percentNAValuesRegions$region)
    
    return(percentNAValuesRegions)
}

pml_write_files = function(predClasse){
    resultsFolder = "./predictionResults"
    
    if (!file.exists(resultsFolder)) {
        dir.create(resultsFolder)
    }
    
    x <- as.character(predClasse)
    
    n = length(x)
    for(i in 1:n){
        filename = paste0("problem_id_",i,".txt")
        write.table(x[i],
                    file=file.path(resultsFolder,
                                   filename),
                    quote=FALSE,
                    row.names=FALSE,
                    col.names=FALSE)
    }
}
